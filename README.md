## How to run this app.?

### Setup kube config env.

Based on kube config file, internal K8s API client can "speak" with your kubernetes cluster.

```bash
export KUBECONFIG=/home/bieli/.kube/config
```

### Compile and run

```bash
cargo run --release
```

or you can build binary

```bash
cargo build --release
```
